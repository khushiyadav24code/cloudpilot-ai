# ☁️ CloudPilot AI

> AI-Powered Autonomous Cloud Infrastructure Provisioning Assistant

## 📌 Project Overview

CloudPilot AI is an AI-powered assistant that converts natural language infrastructure requirements into cloud architecture recommendations and Infrastructure as Code (Terraform).

Example:

User:
"Deploy a React application for 10,000 users."

CloudPilot AI:
- Recommends AWS services
- Generates Terraform
- Estimates cost
- Performs security checks
- Creates architecture diagrams
